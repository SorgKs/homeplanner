# Android приложение HomePlanner

Простое Android приложение для создания событий и управления напоминаниями.

## Структура

Приложение будет создано с использованием:
- Kotlin
- Android SDK
- Retrofit для API взаимодействия
- Room (опционально) для локального кэширования
