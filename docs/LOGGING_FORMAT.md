# Формат логирования в Android приложении HomePlanner

> **Связанные документы:** [OFFLINE_FIRST_ARCHITECTURE.md](OFFLINE_FIRST_ARCHITECTURE.md)

## Обзор

Система бинарного логирования BinaryLogger была полностью удалена из проекта HomePlanner.

Приложение теперь использует стандартное логирование Android (android.util.Log) для отладки и диагностики.
